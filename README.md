# 🎮 Programming Practicum: Semester 2

Сборник проектных задач второго семестра второго курса по дисциплине «Практикум по программированию».

---

## 📂 Структура проекта
* **[Лабораторная работа 1](./Лабораторная%20работа%201/)**
  * `task-1.py` — Скрипт для парсинга новостей с сайта Lenta.ru.
  * `news.txt` — Результат работы первого задания (сохраненные заголовки и ссылки).
  * `task-2.py` — Программа для получения данных о погоде через OpenWeatherMap API.
  * `task-3.txt` — Ответы на контрольные вопросы.
  * `REPORT.md` — Подробный отчет о выполнении заданий и анализе работы.
* [Лабораторная работа 2](./Лабораторная%20работа%202)
    * `task-1.py` — Скрипт для многопоточного парсинга заголовков веб-страниц.
    * `task-2.txt` — Ответы на контрольные вопросы по сетевому взаимодействию и параллелизму.
    * `REPORT.md` — Подробный отчет о выполнении заданий и анализе работы.
* **[Лабораторная работа 3](./Лабораторная%20работа%203/)**
    * `Задание1/` — Скрипт для парсинга с использованием Unit-тестов.
    * `Задание2/` — Клиент OpenWeatherMap с Pytest и использованием Mock-объектов.
    * `Задание3/` — Ответы на контрольные вопросы.
    * `REPORT.md` — Подробный отчет о выполнении заданий и анализе работы.
* **[Лабораторная работа 4](./Лабораторная%20работа%204/)**
    * `Задание1/` — Рефакторинг кода согласно PEP 8 и проверка через Flake8, Black.
    * `Задание2/` — Использование Docstrings (Google Style) и автоматическая генерация документации.
    * `Задание3/` — Ответы на контрольные вопросы.
    * `REPORT.md` — Подробный отчет о выполнении заданий и анализе работы.

* **[docs](https://github.com/Antonova-work/practicum-semester-2/tree/master/docs)** — Исходные файлы технической документации проекта (4 лабораторная):
    * conf.py — Файл конфигурации Sphinx (настройки пути, расширений autodoc, napoleon и темы оформления).
    * index.rst — Главная страница документации, определяющая структуру и содержание разделов.
    * modules.rst — Автоматически сгенерированный файл со списком документируемых модулей.
    * Makefile / make.bat — Скрипты для сборки документации в различные форматы (HTML, PDF).
